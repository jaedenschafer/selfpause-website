## Architecture Recommendation

Use the existing dynamic route pattern (`/affirmations/[slug]/page.jsx`) as a template. Build similar dynamic routes for:

```
app/
  meditation/[slug]/page.jsx      — 7+ pages
  manifestation/[slug]/page.jsx   — 9+ pages
  mindfulness/[slug]/page.jsx     — 8+ pages
  visualization/[slug]/page.jsx   — 3+ pages
  positivity/[slug]/page.jsx      — 4+ pages
  law-of-attraction/[slug]/page.jsx — 3+ pages
  mantras/[slug]/page.jsx         — 2+ pages
  mindset/[slug]/page.jsx         — 3+ pages
```

Each hub gets a `lib/[topic].js` content file (like the existing `lib/affirmations.js`) and a hub index page.

This recovers backlink equity while building a massive content library that targets hundreds of long-tail keywords.
